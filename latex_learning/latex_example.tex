    \subsection{Graphicx}
    Package graphicx can add images as \autoref{fig_1}

    \begin{figure}[htbp] % Where to place the figure: h-Here, t-top of this page, b-bottom of this page, p-top of next page
        \centering % certer image
        \includegraphics[width=\linewidth]{example} 
        \caption{Example image} % image description
        \label{fig_1}
    \end{figure}
    
    Or you can align images in a row like \autoref{fig_2}.

    \begin{figure}[htb]
        \centering
        \begin{minipage}[t]{0.33\linewidth}
            \centering
            \includegraphics[width=\linewidth]{example}
            (a)
        \end{minipage}
        \begin{minipage}[t]{0.33\linewidth}
            \centering
            \includegraphics[width=\linewidth]{example}
            (b)
        \end{minipage}
        \begin{minipage}[t]{0.33\linewidth}
            \centering
            \includegraphics[width=\linewidth]{example}
            (c)
        \end{minipage}
   
        \caption{Example of images in a row.}
        \label{fig_2}
    \end{figure}

    Also you can align images in a matrix like \autoref{fig_3}.

    \begin{figure}[htbp] % two columns of images in a row
        \centering
        \begin{minipage}[htb]{0.45\linewidth} % align images in column
            \centering
            \includegraphics[width=\linewidth]{example}
            (a)
            \includegraphics[width=\linewidth]{example}
            (c)
        \end{minipage}
        \begin{minipage}[htb]{0.45\linewidth}
            \centering
            \includegraphics[width=\linewidth]{example}
            (b)
            \includegraphics[width=\linewidth]{example}
            (d )
        \end{minipage}
        \caption{Example of images in matrix.}
        \label{fig_3}
    \end{figure}
